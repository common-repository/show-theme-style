Links theme style to WordPress Visual Editor



== Description ==

Links theme style to WordPress Visual Editor, thus you can see background style, font style, paddings, margins etc. in your visual editor directly


== Installation ==

1. Upload `show-theme-style` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
